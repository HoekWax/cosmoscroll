Comment jouer ?

Modes de jeu

  • Mode histoire : terminez tous les niveaux jusqu'à l'affrontement avec boss final !
  • Mode arcade : tenez le plus longtemps possible face à des ennemis de plus en plus nombreux.

Commandes clavier

Action Touche
Déplacer le vaisseau
Naviguer dans les menus		 Flèches directionnelles (haut, bas, gauche, droite)
Valider un choix dans les menus Entrée
Utiliser arme 1 Espace
Utiliser arme 2 A
Utiliser bonus Glaçon Contrôle gauche
Mettre en pause/Reprendre la partie P
Quitter immédiatement le jeu Échap

Le jeu peut également être contrôlé avec un joystick ou une manette de jeu.

Les touches du clavier et les boutons du contrôleur peuvent être redéfinis dans les options.

Bonus

  • Health : redonne 1 point de vie au vaisseau
  • Bouclier : redonne 1 point de bouclier
  • Glaçon : donne 1 glaçon. utiliser un glaçon permet d'annuler les effets de la surchauffe, jusqu'à 3 glaçons peuvent être cumulés
  • Missile : donne 1 missile. jusqu'à 3 missiles peuvent être cumulés
  • Triple tir : multiplie par 3 vos tirs pendant 10 secondes !
  • Vitesse : multiplie par 2 votre vitesse pendant 10 secondes
  • Stoned : Vous ne savez plus très bien piloter pendant 10 secondes...
  • Banane magique : invincibilité pendant 10 secondes

Note : il n'y a pas de cheat codes dans ce jeu.

Comment retrouver la configuration d'origine ?

Si vous vous voulez retrouver les paramètres par défaut de l'application, effacez le fichier config/config.txt, un nouveau fichier sera créé automatiquement.
(Attention, votre ancienne configuration sera perdue)

Comment compiler le jeu depuis les sources ?

CosmoScroll compile sous les plateformes GNU/Linux (GCC) et Windows (mingw32)
Les autres systèmes et compilateurs n'ont pas été testés.

Vous devez tout d'abord installer les bibliothèques suivantes :

  • SFML, version 1.4
  • libdumb

Le moyen le plus simple de récupérer les sources est de faire un checkout sur le serveur SVN du projet (reportez-vous aux instructions de la page).

Un fichier projet Code::Blocks multi-plateformes (cosmoscroll.cbp) est fourni et permet de compiler le jeu depuis l'IDE.
Un Makefile UNIX est également fourni avec les sources du projet (tapez simplement "make" en console) afin de compiler le jeu sans passer par un IDE.
