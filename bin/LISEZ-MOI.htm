Comment jouer ?

Modes de jeu

  • Mode histoire :
    Terminez tous les niveaux jusqu'à l'affrontement avec le boss final et débloquez le mode hardcore !
  • Mode arcade :
    Tenez le plus longtemps possible face à des ennemis de plus en plus nombreux et de plus en plus puissants.
    Tentez d'établir un nouveau record et envoyer le en ligne afin de vous mesurer aux autres joueurs !

Le jeu peut également être contrôlé avec un joystick ou une manette de jeu.

Les touches du clavier et les boutons du contrôleur peuvent être redéfinis dans les options.

Bonus

  • Point de vie : Donne +1 point de vie au vaisseau.
  • Bouclier : Donne 1 point de bouclier.
  • Glaçon : Donne +1 Glaçon. Utiliser un glaçon permet d'annuler les effets de la surchauffe, jusqu'à 3 glaçons peuvent être cumulés.
  • Missile : Donne +1 Missile. Jusqu'à 3 missiles peuvent être cumulés. À utiliser quand les ennemis sont trop nombreux !
  • Double tir : double vos tirs pendant 10 secondes.
  • Triple tir : triple vos tirs pendant 10 secondes.
  • Vitesse : multiplie par 2 votre vitesse pendant 10 secondes
  • Super Banane : max points de vie, max bouclier, chaleur à 0, +1 Glaçon, +1 Missile

Comment retrouver la configuration d'origine ?

Si vous voulez retrouver les paramètres par défaut de l'application, effacez le fichier config/config.cfg, un nouveau fichier sera créé automatiquement à la prochaine utilisation du jeu.
Attention, en effaçant ce fichier, votre ancienne configuration (scores, progression dans les niveaux, préférences, etc.) sera perdue.

Comment compiler le jeu depuis les sources ?

CosmoScroll compile sous les plateformes GNU/Linux (GCC), Mac OS (GCC) et Windows (mingw32).
Les autres systèmes et compilateurs n'ont pas été testés.

Vous devez tout d'abord installer les bibliothèques suivantes :

  • SFML, version 1.6
  • libdumb (pour Debian/Ubuntu, installer le paquet libdumb1-dev)

Le moyen le plus simple de récupérer les sources est de faire un checkout sur le serveur SVN du projet (reportez-vous aux instructions de la page).

Un fichier projet Code::Blocks multi-plateformes (cosmoscroll.cbp) est fourni et permet de compiler le jeu depuis cet IDE.
Un Makefile UNIX est également fourni avec les sources du projet (tapez simplement make en console) afin de compiler le jeu sans passer par un IDE.

J'ai trouvé un bug / Je veux soumettre une idée

Si vous pensez avoir trouvé un dysfonctionnement dans CosmoScroll, merci de le rapporter à cette adresse (si le bug n'est pas encore répertorié).
Une description permettant de reproduire le problème serait également appréciée.

Vous pouvez aussi contacter l'un des auteurs de CosmoScroll si vous avez une suggestion d'amélioration, elle sera la bienvenue !
